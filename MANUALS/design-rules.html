Physical Design

Design rules for Fab Vine parts.

This manual will define how physical modules should be designed so parts can be fabricated, assembled, repaired, and shared across the Fab Lab network.

Basic Rule

Design parts as modular, readable components. A contributor should be able to understand what a part does, where it connects, and how it can be fabricated without needing hidden context.

Topics To Document

  • Snap-fit and connector geometry.
  • Module scale and tolerances.
  • Material assumptions.
  • File naming and versioning.
  • How design files connect to assembly guides.
