Programming progress

V3 LED interaction studies

Sprint 5 includes a sequence of testable programs for the 40-pixel body strip. The strip is physically arranged as four groups of ten pixels on a twisted cylinder. Every group is indexed from bottom to top. Each program remains in its own folder so a tested behavior can be loaded again without overwriting the others.

VersionPurposeWhen to use itFiles
V3.3 Ascending Breath A purple, blue, and green breath rises through all four groups. Check bottom-to-top wiring and the first independent-pixel animation. Guide · Arduino
V3.4 Organic Breath Adds natural inhale, hold, exhale, rest, and a soft trailing glow. Evaluate a less mechanical bottom-to-top breathing rhythm. Guide · Arduino
V3.5 Center Out Breath Starts at the two center LEDs and expands toward both ends. Current aesthetic breathing reference. Guide · Arduino
V3.6 Direction Calibration Identifies each colored ten-pixel path and its physical orientation. Run after assembling a new twisted cylindrical body. Guide · Arduino
V3.7 Neighbor Compass Uses animated directions to show faces that still have no neighbor. Diagnose face recognition and guide module placement. Guide · Arduino
Important calibration reference

Direction Calibration color map

V3.6 is the physical reference for all later directional behavior. Place the module upright and look at it from the display/front side. The program lights one ten-pixel group at a time and moves a bright marker from bottom to top.

ColorPixelsConfirmed faceExpected movement
Red0–9FrontBottom to top
Green10–19RightBottom to top
Blue20–29Back / posteriorBottom to top
Purple30–39LeftBottom to top

These colors identify physical sides; they are not error states. If a future strip is mounted in another orientation, repeat this calibration and correct the group mapping before loading Neighbor Compass.

Neighbor Compass visual language

  • Front: the red group moves upward.
  • Right: the green group moves upward.
  • Back / posterior: the blue group moves upward.
  • Left: the purple group moves upward.
  • Top: white light moves upward through all four groups.
  • Bottom: white light moves downward through all four groups.

V3.7 cycles only through faces without a recognized neighbor. Connecting a face removes that direction after the neighbor timeout; disconnecting it makes the direction return. When all faces are connected, the strip returns to organic breathing.

How to load a program

  1. Open the version folder and then open the matching .ino file.
  2. In Arduino IDE, select XIAO ESP32C6 and the module's serial port.
  3. Keep the companion src/PacketCommunication.h file inside the same version folder.
  4. Install Adafruit GFX, NeoPixel, NeoMatrix, and SSD1306 libraries if Arduino reports them missing.
  5. Compile and upload. The 40-pixel body strip data line is connected to D10.
