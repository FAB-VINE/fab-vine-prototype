Node Logic / V1

V1 is the first unified Fab Vine node firmware.

All nodes are currently expected to run the same base firmware. Differences should be handled through configuration or future versions, not separate sketches for each node.

What V1 Contains

  • OLED expression display behavior.
  • Places for sensor logic, LED behavior, and network communication.
  • A local preview tool for OLED expressions.
  • An Arduino sketch tested on XIAO ESP32-C6.
