Sprint 3 Pinout

Sprint 3 changes the neighbor system from detect-to-ground contacts to active TX/RX signaling. The module sends presence from one common TX pin and listens through one RX pin per face.

Face Connector Logic

Every PCB can be identical if each face connector is physically mirrored so that TX lands on RX, RX lands on TX, and ground lands on ground. 

Each face carries:

GND
TX_COMMON = D6
RX_FACE_n = unique face RX pin

When two modules connect:

Module A GND <----> Module B GND
Module A TX  ----> Module B RX
Module A RX  <---- Module B TX

Firmware Compatibility

The current tested firmware is still Sprint 2. It uses D8 as the shared data/signal line and uses detect-to-ground inputs on D0,D1,D2,D3,D6,D7. Sprint 3 requires a firmware update before testing this PCB pinout on physical modules.

A separate communication experiment also exists in NODE-LOGIC/fab-vine-node/V1/firmware/neighbor-communication/; it is a prototype reference, not the current production node firmware.
