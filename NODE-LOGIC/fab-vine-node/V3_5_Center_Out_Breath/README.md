# V3.5 Center Out Breath

Organic breathing that begins at the physical center of every ten-pixel group and expands toward both ends. Because each group has an even number of pixels, positions 4 and 5 form the visual center.

The four groups breathe together. The animation keeps the V3.4 inhale, hold, exhale, and rest timing while changing the spatial direction from vertical travel to center-out expansion.

## Use

Open `V3_5_Center_Out_Breath.ino`, select `XIAO ESP32C6`, and upload it. Connect the 40-pixel body strip to `D10`.

Use this version as the current aesthetic breathing reference when no connection-direction diagnostic is needed.
