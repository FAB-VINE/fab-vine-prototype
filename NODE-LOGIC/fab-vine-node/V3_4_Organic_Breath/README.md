# V3.4 Organic Breath

Refines V3.3 into an asymmetric, more natural breathing rhythm. The animation inhales, briefly holds, exhales, and rests instead of moving with a perfectly mechanical pulse.

A bright front and two softer trailing layers move upward through each ten-pixel group. Purple, blue, and green are blended across both time and height.

## Use

Open `V3_4_Organic_Breath.ino`, select `XIAO ESP32C6`, and upload it to the module. The 40-pixel strip uses `D10`, with four groups of ten pixels ordered bottom to top.

Use this version when evaluating the most natural bottom-to-top breathing behavior without directional neighbor guidance.
